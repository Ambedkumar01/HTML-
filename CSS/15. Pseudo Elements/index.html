This is a demonstration of CSS Pseudo Elements.
CSS Pseudo Elements are used to style specific parts of an element.
For example, you can use them to add content before or after an element, or to style the first letter or line of a paragraph.
In this example, we will explore how to use the ::before and ::after pseudo elements to add decorative content to our HTML elements.
